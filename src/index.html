    HTML

    Это язык гипертекстовой разметки текста. Он нужен, чтобы размещать на веб-странице элементы: текст, картинки, таблицы и видео
    CSS3

    Язык стилей CSS преображает сайт и придает ему привлекательный и эффектный внешний вид
    JavaScript ES6+

    Язык программирования высокого уровня, использующийся для придания интерактивности веб-страницам
    jQuery

    Это набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML
    Bootstrap

    Это открытый и бесплатный HTML, CSS и JS фреймворк, который используется веб-разработчиками для быстрого создания адаптивных дизайнов сайтов
    PHP

    Это язык программирования, который создавали для разработки веб-приложений, но позже он эволюционировал в язык общего назначения
    Vue.js

    JavaScript-фреймворк с открытым исходным кодом для создания пользовательских интерфейсов. Легко интегрируется в проекты с использованием других JS-библиотек
    WordPress

    Это свободно распространяемая система управления содержимым сайта, написана на PHP. Сфера применения — от блогов до достаточно сложных новостных ресурсов
